## Beta notice

WemX v3 is currently in **beta** and is **not intended for production use yet**. Use it for testing, exploring new features, and reporting issues before the stable release.

## What's new

- **Full rewrite** — Rebuilt from scratch with a modern codebase
- **Source-available** — Source on GitHub for transparency and community contributions ([LICENSE.md](LICENSE.md))
- **Scalable architecture** — Flexible database structure designed for growth
- **Livewire powered** — Reactive, fast admin and client interfaces
- **Multi-machine support** — Built for load-balanced deployments

Other highlights include a **cart system**, **multi-server support**, **multi-gateway payments**, **tax collection**, **audit logs**, extensions, and more.

## Tech stack

- PHP 8.3+
- Laravel 12
- Livewire 3 & Volt
